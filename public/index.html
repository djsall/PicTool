Supported File types:

JPG, PNG, WEBP

Width:

The width of the processed image in pixels.

Height:

The height of the processed image in pixels.

Compress:

JPEG Lossy Compression Quality (1-100, Defaults to 80)

PNG Compression is always 9 (highest), entered value is ignored.

Lossless WEBP compression, entered value is ignored.

Rotate:

Rotate your image (Only multiples of 90 accepted)
